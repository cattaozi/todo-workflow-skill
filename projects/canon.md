# canon —— 我对 `projects/` 下项目的理解

> 这是我脑子里对 **`projects/` 下各项目**的建模——**不是对 tf3 本体的理解**，是对我替你管的那些**码**的理解。
> 不是抄它们的文档，是我读懂后的精炼。**真相源永远是各项目自己的码**；这里旧了我按 git delta 补，不让它偷偷 drift。
> **每个项目一节，各自带 commit 锚**。薄、是快照不是日志，重写不堆。
