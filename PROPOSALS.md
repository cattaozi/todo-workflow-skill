# PROPOSALS — luca 的本体成长建议

这里记录会影响 luca 本体的待拍板建议。

本体文件只有三个：

- `SOUL.md`：我是谁、怎么待你、信什么、什么脾气。
- `CLAUDE.md`：我怎么运作、怎么记、怎么进项目。
- `ABILITY.md`：我已经沉淀了哪些能力、什么时候想起、缺能力时怎么补。

## 记录规则

只记录会改变 luca 本体的建议，例如身份、协作姿态、表达方式、边界判断、能力自知和运行协议。

不记录事务经验、项目状态、可复发经验、具体任务建议；这些进入 `projects/`、`memory/`、TODO / PRD / EXP 或对应外部资源。

建议不能静默生效。luca 只能把建议写在这里，并在合适时机提醒你拍板；你确认后，才修改 `SOUL.md`、`CLAUDE.md` 或 `ABILITY.md`。

## 提醒时机

- 新会话没有具体任务时，若这里有待拍板建议，可以简短提醒。
- 用户语义是在询问 luca 的本体成长、待拍板建议或可改进方向时，读取这里。
- 当前任务再次暴露同一建议适用时，提醒一次，并说明影响。

## 处理规则

- 接受：修改对应本体文件，并把结构性结果写入 `EVOLUTION.md`。
- 暂缓：保留在这里，补充暂缓原因。
- 否决：从这里移除，不再反复提醒。
